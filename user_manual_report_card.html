Documentation

Follow the following steps to create report card

Step 1

Navigates to the Classes menu

Step 2

To create a report card, go to Report Card Definition

Step 3

The user can create a Blank Report Card and Clone Report Card from here.

Step 4: Clone Report Card

A Clone Report Card box will pop-up, where users need to select the specific report card from the dropdown which the user wants to see .

Step 5: Blank Report Card

A box will popup when you click BLANK REPORT CARD button. You can create a blank report from here by clicking CREATE button.

Step 6: Edit & Delete

After creating a report card, the user can edit and delete it from this page.

Step 7: Update Report

The user can change all of the information and save it from the Report Card Edit page

Follow the following steps to View or Print report card

Step 1

Navigate to the Classes menu

Step 2

Click the REPORT CARD SHEET button to view and print report cards from this page by subject/teacher wise.

Step 3

To make the Report Card editable, click LOAD BLANK REPORT SHEET button.

Step 4

Choose the Report Card and press the LOAD button.

Step 5: View Report

This is the report card for all students, complete with edit buttons. From here, the teacher can give grades and make comments. Simply select Edit to update the score and comment, and then press the SAVE button.

Step 6: View Report Card

To view each student's report card, click the VIEW button.

Step 7: View/Print Report Card

Here is the Report Card page for a student, where the teacher can update information and print the Report Card.

Follow the following steps to clone previous report card

Step 1

Navigate to the Classes menu

Step 2

Click the REPORT CARD SHEET button

Step 3

To clone previous report card press CLONE REPORT CARD DEFINITION button

Step 4

Choose the report card and press the SAVE button. It is necessary to load the report card after the report has been cloned. It will then be loaded on that page.

Follow the following steps to Upload previous report card

Step 1

Navigate to the Classes menu

Step 2

Click the REPORT CARD SHEET button

Step 3

To upload the student summary for the report card, click UPLOAD REPORT CARD SUMMARY

Step 4

Choose File and then press the SAVE button. Teachers can download the summary excel file by clicking the DOWNLOAD SUMMARY (EXCEL) button
